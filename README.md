# 💰 Personal Finance Manager (Full Stack Project)

A Full Stack Web Application developed to manage personal income and expenses efficiently.

This project demonstrates my skills in Java Full Stack Development using Spring Boot and React.

---

## 👩‍💻 About the Project

Managing personal finances manually can be difficult.  
This application helps users track their income and expenses in a simple and organized way.

Users can:

✔ Add income records  
✔ Add expense records  
✔ View all transactions  
✔ Manage financial data efficiently

## ⚙️ How to Run Locally

### 🔹 Backend

1. Open Backend folder in IDE
2. Configure database in `application.properties`
3. Run Spring Boot application

Runs on:
http://localhost:8088

---

### 🔹 Frontend

1. Open frontend folder in VS Code
2. Run:

npm install  
npm start  

Runs on:
http://localhost:3000
